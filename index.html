MINHAS HABILIDADES

HTML

HTML (HyperText Markup Language) estrutura o conteúdo das páginas web usando tags para definir títulos, parágrafos e imagens.

CSS

CSS (Cascading Style Sheets) controla a aparência das páginas web, permitindo ajustar cores, fontes e layout dos elementos.

JavaScript

JavaScript é uma linguagem de programação que adiciona interatividade às páginas web, como animações e manipulação de eventos.

Projetos

Scratch Objetivos e Metas



Meu projeto no Scratch foi desenvolvido para um curso e tem o objetivo de demonstrar como definir e alcançar metas de maneira visual e interativa. O projeto utiliza setas que, quando seguidas, mudam o cenário, ilustrando a progressão através de diferentes objetivos. Através dessas mudanças de cenário, o projeto destaca a importância do planejamento e da orientação estratégica para atingir metas de forma eficaz..

Planejamento Semanal



Meu projeto em HTML e CSS foi desenvolvido para o planejamento semanal de atividades. Ele consiste em uma interface simples e intuitiva onde os usuários podem registrar e marcar suas tarefas concluídas. O objetivo do projeto é fornecer uma ferramenta prática para organização pessoal, facilitando o acompanhamento do progresso e a gestão eficiente do tempo.
